# Question Answering System

## Setup Instructions

1. Install dependencies:
    ```sh
    pip install -r requirements.txt
    ```

2. Run the system:
    ```sh
    streamlit run app.py
    ```

## Dependencies

- spacy
- sentence-transformers
- faiss-cpu
- transformers
- streamlit
